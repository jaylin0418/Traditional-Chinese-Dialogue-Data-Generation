# IMPORTANT: IndexTTS2 Installation Instructions
# ═══════════════════════════════════════════════════════════════════════════
# IndexTTS2 requires manual installation via git clone and uv package manager:
#
# 1. Install uv package manager:
#    pip install -U uv
#
# 2. Clone IndexTTS repository:
#    git clone https://github.com/index-tts/index-tts.git
#    cd index-tts
#    git lfs pull
#
# 3. Install dependencies:
#    uv sync --all-extras
#    (On Windows, remove --all-extras if DeepSpeed installation fails)
#
# 4. Download the model (choose one):
#    # Via HuggingFace:
#    uv tool install "huggingface-hub[cli,hf_xet]"
#    hf download IndexTeam/IndexTTS-2 --local-dir=checkpoints
#
#    # Via ModelScope:
#    uv tool install "modelscope"
#    modelscope download --model IndexTeam/IndexTTS-2 --local_dir checkpoints
#
# 5. Make sure to have speaker reference audio files (.wav) in the 'examples' directory
#    You need at least 2 different speaker audio files for dialogue generation
#
# 6. Update your config to point to:
#    - model_dir: path to checkpoints directory (default: "checkpoints")
#    - spk_audio_dir: path to speaker audio samples (default: "examples")
#
# For more information, visit: https://github.com/index-tts/index-tts
